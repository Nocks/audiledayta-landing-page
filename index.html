AudibleDayta

This is a personal project of Enock Kwesi Addey. The mission of this project is: "Bringing to light and hearing clearly what your data has to say".

To start with, the project will showcase some data analysis findings on some of the interesting open data out there.

Over time, AudibleDayta will render various data science services to startups and SMEs, while contributing to innovative open-source data science projects.
